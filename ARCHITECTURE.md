# Architecture

## Core Concepts

- PCS (Pitch-Class Set): {0–11}
- Superset: (k, root)
- Subset: chord derived via offsets

## Generators

k=7 → 0,2,4,5,7,9,11  
k=6 → 0,2,4,6,8,10  
k=5 → 0,2,4,7,9  
k=8 → 0,2,3,5,6,8,9,11  
k=4 → 0,3,6,9  
k=3 → 0,4,8  

## Offsets

Triads: 0,2,4  
Sus4: 0,3,4  
7ths: 0,2,4,6
